# Sales Analytics End-to-End Project

## Overview

This project presents a complete sales analytics workflow using SQL, Python, and Power BI.

The objective was to analyze sales performance, customer behavior, product categories, and regional revenue to identify key business opportunities.

---

## Project Workflow

1. Data extraction and KPI analysis in SQL
2. Data cleaning and transformation in Python
3. Data visualization in Python
4. Interactive dashboard creation in Power BI
